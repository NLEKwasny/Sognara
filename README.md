# Sognare Pizza

Strona mobilnej pizzerii neapolitańskiej — live cooking na weselach, urodzinach i eventach.
Jednoplikowa, responsywna strona (HTML/CSS/JS, bez bibliotek) z dwukolumnowym hero,
galeria 3D flip, kalendarzem dostepnosci i formularzem rezerwacji podlaczonym do Supabase.

Otworz `index.html` w przegladarce lub udostepnij przez GitHub Pages.
